Classifying Heart Disease with Logistic Regression
Logistic Regression Modeling in Python

I worked with a real-life dataset : the Heart Disease Data Set from the UCI Machine Learning Repository. This dataset comes from the famous Cleveland Clinic Foundation, which recorded information on various patient characteristics, including age and chest pain, to try to classify the presence of heart disease in an individual. This a prime example of how machine learning can help solve problems that have a real impact on people's lives.
Stochastic Gradient Descent on Linear Regression

I decided to build a predictive model using Machine Learninhg, especifically a linear regressor using Stochastic Gradient Decsent.
